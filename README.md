# 学生宿舍管理系统 java649

## 项目概述

学生宿舍管理系统是基于SpringBoot+Vue的前后端分离架构，致力于提升学校宿舍管理的效率与资源分配合理性，简化管理流程。

## 技术栈

- **服务端技术**：SpringBoot、MyBatis Plus
- **前端技术**：Vue 3.x、Element Plus、wangeditor、axios、echarts
- **运行环境**：JDK 1.8、MySQL 5.7、Node.js 14.21.3、IntelliJ IDEA、Maven

## 功能模块

系统主要包括以下功能模块：

- 用户管理
- 宿舍管理
- 信息管理
- 申请管理
- 访客管理
- 个人信息

## 系统角色

系统定义了以下三种角色：

- **系统管理员**：拥有最高权限，可访问所有管理菜单。
- **宿舍管理员**：负责宿舍相关的管理工作。
- **学生**：可进行信息查询和申请操作。
